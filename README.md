# TradSig

交易记录与实盘更新提醒工具（iOS + Web）

TradSig 是一款用于 **交易过程记录、复盘与实盘状态提醒** 的工具型产品。  
系统以网站服务为核心，配合 iOS 客户端，在有新的交易记录或状态更新时，通过推送及时提醒用户查看。

---

## 核心功能

- 📊 交易记录展示（按品种 / 页面维度）
- 🔔 实盘更新提醒（Apple 推送通知）
- 📲 推送点击承接到对应记录页面
- 📱 轻量级 iOS 客户端（WebView + 原生推送能力）

---

## 产品形态

- **Web 端**  
  负责登录、权限控制、交易记录展示与数据管理

- **iOS 客户端**  
  用于展示网站内容，并承接系统推送，实现更及时的提醒体验

---

## 适用场景

- 交易过程记录与回顾  
- 实盘更新提醒，避免错过关键记录  
- 个人复盘与行为管理使用  

> 本产品不提供任何投资建议，仅用于记录与展示交易过程。
